# Browser RegEdit

A web-based simulation of the Windows Registry Editor built with Python (Flask) and Vanilla JavaScript.

## Project Structure

- **run.py**: Entry point to start the server.
- **app/**: Main application package.
    - **registry.py**: Handles backend data logic (The mock registry).
    - **routes.py**: Handles web requests.
    - **static/**: CSS and JavaScript files.
    - **templates/**: HTML files.
